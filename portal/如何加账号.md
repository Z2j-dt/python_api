# 门户怎么加登录账号（方式一）

## 第一步：在「项目根目录」生成密码哈希

在 **E:\python_api** 下打开命令行（PowerShell 或 CMD），**先激活虚拟环境**（如果还没激活）：

```powershell
cd E:\python_api
venv\Scripts\activate
```

然后执行下面**其中一种**方式。

### 方式 A：用脚本（推荐）

```powershell
python portal/add_user.py
```

按提示输入：**用户名**、**密码**、**权限模块**，脚本会输出一段配置，你复制到 `auth_config.py` 里即可。

### 方式 B：手动生成哈希

在项目根、已激活 venv 的情况下执行（把 `admin123` 换成你要的密码）：

```powershell
python -c "from werkzeug.security import generate_password_hash; print(generate_password_hash('admin123'))"
```

会输出一长串，例如：`scrypt:32768:8:1$xxxxx$yyyyy`，**整段复制**下来。

---

## 第二步：新建并编辑 auth_config.py

1. 在 **portal** 目录下，把 **auth_config.example.py** **复制一份**，并改名为 **auth_config.py**（没有 example）。
2. 用记事本或 VS Code 打开 **portal/auth_config.py**。
3. 按下面说明改内容。

### 如果你用了方式 A（add_user.py）

- 脚本最后会输出一段类似这样的内容：
  ```python
  "zhangsan": {
      "password_hash": "scrypt:32768:8:1$...",
      "modules": ["hive_metadata", "sql_lineage"],
  },
  ```
- 把这段**整段复制**，粘贴到 `AUTH_USERS = {` 后面、最后一个 `}` 前面（注意逗号），保存。

### 如果你用了方式 B（手动生成哈希）

- 在 `AUTH_USERS` 里加一个用户，例如：
  ```python
  "新用户名": {
      "password_hash": "这里粘贴第一步复制的那一长串",
      "modules": ["hive_metadata", "sql_lineage", "sr_api"],
  },
  ```
- **modules** 表示能看哪些模块，三选一或组合都可以：
  - `"hive_metadata"` = 数据字典
  - `"sql_lineage"` = 数据血缘
  - `"sr_api"` = 业务场景
- 保存文件。

---

## 第三步：部署并重启

1. 把改好的 **portal/auth_config.py** 传到服务器（和现在传代码一样，放到服务器上的 `portal/` 目录）。
2. 在服务器上执行：
   ```bash
   supervisorctl restart portal
   ```
3. 用新账号登录门户页面试一下。

---

## 示例：最终 auth_config.py 长什么样

下面是一个示例（密码哈希是假的，你要用自己的）：

```python
# -*- coding: utf-8 -*-
# 不要提交到版本库

AUTH_USERS = {
    "admin": {
        "password_hash": "scrypt:32768:8:1$abc123$def456...",
        "modules": ["hive_metadata", "sql_lineage", "sr_api"],
    },
    "zhangsan": {
        "password_hash": "scrypt:32768:8:1$xyz789$...",
        "modules": ["hive_metadata", "sql_lineage"],
    },
}
```

保存为 **portal/auth_config.py**，重启 portal 后，就可以用 admin 和 zhangsan 两个账号登录。
